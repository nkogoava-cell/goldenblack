# goldenblack
“GoldenBlack est un projet K-pop imaginatif créé par une jeune autrice-compositrice. Ce site présente l’univers du groupe : mélodies IA inédites, moodboards visuels, ambiance sombre et néon, et un espace fanclub pour rejoindre la communauté dès maintenant. Le futur de la K-pop commence ici.”
